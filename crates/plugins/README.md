# doing-plugins

Export and import plugins for the [doing](https://github.com/aaronmallen/doing) CLI.

This crate provides the plugin system for exporting entries to various formats and importing entries from external
sources.

## Export Plugins

- `byday` тАФ group entries by day
- `csv` тАФ comma-separated values
- `dayone` тАФ Day One journal format
- `doing` тАФ native doing file format
- `html` тАФ HTML with CSS styling
- `json` тАФ JSON export
- `markdown` тАФ Markdown format
- `taskpaper` тАФ TaskPaper format
- `timeline` тАФ chronological timeline view

## Import Plugins

- `calendar` тАФ iCalendar (ICS) files
- `doing` тАФ doing file format
- `json` тАФ doing JSON export format
- `timing` тАФ Timing.app JSON reports
